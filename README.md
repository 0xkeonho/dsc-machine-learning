## Silabus Pembelajaran

Program ini terdiri dari 8 pertemuan yang disusun secara bertahap:

### Pertemuan 1: Pengantar Machine Learning & Basic Math
* **Pengantar Machine Learning:** Penjelasan teori dasar mengenai Supervised, Unsupervised, dan Reinforcement Learning (RL). Pemahaman perbedaan fundamental antara konsep Regresi dan Klasifikasi, serta urgensi pembagian data menjadi Training dan Testing.
* **Basic Math (Matematika Dasar):**
    * **Aljabar Linear:** Pemahaman konsep Eigenvalue dan Eigenvector.
    * **Kalkulus:** Dasar-dasar turunan dan turunan parsial.
    * **Statistika Dasar:** Konsep dasar regresi dan penurunan rumus Maximum Likelihood Estimation (MLE) untuk Linear Regression secara matematis ($\hat{\beta} = (X^T X)^{-1} X^T Y$).

### Pertemuan 2: Data Cleaning & Exploratory Data Analysis (EDA)
Sesi praktik langsung (hands-on) yang difokuskan pada persiapan data:
* **Data Cleaning:** Praktik teknik imputasi nilai yang hilang (missing values) menggunakan mean, median, atau modus. Meliputi juga penghapusan data duplikat serta standarisasi format data.
* **Exploratory Data Analysis (EDA):** Pengenalan berbagai fungsi plot dan visualisasi untuk menggali informasi serta menganalisis karakteristik data.

### Pertemuan 3: Feature Engineering & Preprocessing
Fokus pada rekayasa dan prapemrosesan fitur agar data siap digunakan oleh model:
* **Feature Engineering:** Penerapan teknik pembuatan fitur baru yang didasarkan pada pemahaman domain knowledge.
* **Feature Encoding:** Penjelasan alasan data kategorikal harus melalui proses encoding beserta jenis-jenis encoding yang umum digunakan.
* **Feature Scaling:** Penjelasan mengenai penyeragaman rentang nilai fitur (scaling) untuk optimalisasi model.

### Pertemuan 4: Klasifikasi, Imbalance Class, dan Pengenalan Berbagai Model
* **Klasifikasi & Logistic Regression:** Penjelasan konsep dasar klasifikasi sebagai Linear Regression yang ditambah fungsi sigmoid, serta perbedaan antara klasifikasi biner dan multiclass.
* **Imbalance Class:** Cara mengidentifikasi dan menangani masalah ketidakseimbangan kelas dalam dataset.
* **Pengenalan Jenis Model:** Implementasi pengkodean untuk algoritma Support Vector Machine (SVM), KNN, Decision Tree, dan Random Forest baik untuk kasus regresi maupun klasifikasi.

### Pertemuan 5: Algoritma Dasar, Evaluation Metric, & Hyperparameter Tuning
* **Konsep Algoritma:** Penjelasan logika dan cara kerja algoritma dasar seperti KNN dan Decision Tree.
* **Evaluation Metric:** Penggunaan berbagai metrik evaluasi untuk mengukur performa model.
* **Hyperparameter Tuning:** Cara pencarian parameter optimal menggunakan metode Grid Search atau Random Search, serta contoh implementasi menggunakan Optuna.

### Pertemuan 6: Aplikasi Basic Math pada Machine Learning
Menjelaskan aplikasi nyata dari materi matematika dasar pada algoritma:
* **Principal Component Analysis (PCA):** Penerapan menggunakan konsep Eigenvalue dan Eigenvector.
* **Gradient Descent:** Penerapan menggunakan konsep turunan dalam optimasi.

### Pertemuan 7: Dasar-dasar Neural Network
Membangun fondasi pemahaman sebelum masuk ke arsitektur jaringan saraf tiruan:
* **Loss Function & Cost Function:** Perbedaan antara Loss Function dan Cost Function untuk kasus regresi maupun klasifikasi.
* **Implementasi Manual:** Praktik membangun model Linear Regression dari awal menggunakan algoritma Gradient Descent.

### Pertemuan 8: Membangun Neural Network
Gabungan teori matematika dan pemrograman dalam skala kompleks:
* **Membangun Model NN:** Penjelasan cara membangun model Neural Network secara komprehensif, baik dari sisi matematis menggunakan Chain Rule pada kalkulus maupun dari sisi eksekusi pengkodean.
